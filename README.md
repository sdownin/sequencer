## Getting Started

- Download zipped file
- Unzip, keeping all files inside same directory
- Run app by executing runfile: `run.vbs`

```
SequenceR/
    |--GoogleChromePortable/
    |--logs/
    |    └--SequenceRlog.txt
    |--R-Portable/
    |--shiny/
    |    └--www/
    |        └--js/
    |            └--main.js
    |    |--analysisModule.R
    |    |--app.R
    |    |--appServer.R
    |    |--appUi.R
    |--TestData/
    |    |--test_alphabet.csv
    |    |--test_data.csv
    |    └--test_subcostmat.csv
    |--run.vbs
    └--runShinyApp.R
```
